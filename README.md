# 🛰️ Exo-Pulsar | Master Dashboard 🛰️

### **The Next-Generation Virtual Classroom & Meeting Workspace**
Exo-Pulsar is a high-fidelity, premium SaaS-grade meeting platform designed for maximum productivity and immersive collaboration. Built with **Glassmorphism aesthetics**, **PS4-style dynamic backgrounds**, and **AI-powered assistant features**.

## ✨ Key Features
*   🎨 **Appearance Customization:** 12+ premium theme colors, custom hex picker, and 5 professional typography options.
*   🌊 **PS4-Style Animated Backgrounds:** 12 unique animated gradient themes with interactive particle effects.
*   🌙 **Total Dark Mode:** 100% coverage across whiteboard, calendar, chat, and all dashboard elements.
*   🎨 **Interactive Whiteboard:** Real-time collaborative drawing with multiple pen colors, sizes, and an eraser.
*   📅 **Academic Calendar:** Full event planner with date-specific modals and school-year tracking.
*   🤖 **AI Meeting Assistant:** Auto-generated summaries, sentiment analysis, and experimental voice commands.
*   🎭 **Advanced Video Filters:** AI Studio lighting, AR masks, and DSLR-style background blur.

---

## 🛠️ The Tech Stack
*   **Core:** HTML5 / CSS3 / ES6 Javascript
*   **Design:** Vanilla CSS Design System (Custom variables/tokens)
*   **Real-time:** PeerJS / WebRTC for video and P2P communication
*   **Animations:** CSS Keyframes & Canvas-based Particle Engine
